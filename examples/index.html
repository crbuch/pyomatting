Closed-Form Alpha Matting using Pyodide

Instructions:

  • Images: Upload one or more source images you want to extract the foreground from
  • Trimaps: Upload the same number of trimaps where:
    • Black (0) = definitely background
    • White (255) = definitely foreground
    • Gray (128) = unknown region to be computed
  • Each image should have a corresponding trimap with the same dimensions
  • The algorithm will process all images in batch and compute alpha mattes for the unknown regions
